__SIMPLE NODE SWAGGER DOC EXAMPLE__

___PROJECT STRUCTURE___
 - This project uses express and swagger js doc, the source files are in `app` folder

___INSTALL___
```
cd app
npm install
    or
cd app
yarn install
```
___RUN___
```
cd app
npm start
   or
cd app
yarn start
```
